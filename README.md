# EcoCharge Dublin: Data-Driven Dashboard

An interactive urban planning and decision-support system designed to optimise EV charging infrastructure and renewable energy integration across Dublin.

---

## Current Status

All major components are connected end-to-end. The dashboard pulls live data from the backend API and displays it in real time.

---

## Frontend Features

### Map & Layers
- Interactive Dublin base map with Standard and Dark Mode options
- Layer toggles for ESB Networks, EasyGo Networks, and Other Networks (Tesla, etc.)
- Clickable markers with address, operator, and charger count

### Traffic Heatmap
- Live traffic volume heatmap sourced from SCATS data via `/api/traffic`
- Colour gradient from blue (low) to red (high) represents EV charging demand intensity
- 223 traffic monitoring sites across Dublin

### AI Recommendation Layer
- Top 10 recommended new EV charging locations generated by K-Means clustering
- Each site displayed as a numbered marker ranked by supply-demand gap score
- Clicking a marker shows rank and gap score

### Dashboard Panel
- Live statistics: current charger count, recommended sites, traffic sites
- Top 5 priority sites listed with gap scores

---

## Backend API Endpoints

| Endpoint | Description |
|---|---|
| `GET /api/chargers` | All 134 Dublin EV charging stations (GeoJSON) |
| `GET /api/recommendations` | Top 10 recommended new sites ranked by gap score |
| `GET /api/energy/latest` | Latest renewable energy data from EirGrid |
| `GET /api/traffic` | SCATS traffic volume data (223 sites) |

---

## Tech Stack

**Frontend:** React, Leaflet.js, react-leaflet, leaflet.heat  
**Backend:** FastAPI, PostgreSQL (Supabase), SQLAlchemy  
**Data:** Smart Dublin, EirGrid, ESB eCars, OSM
